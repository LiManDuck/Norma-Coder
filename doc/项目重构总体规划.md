# 项目总体规划 

NormaCoder将作为一个轻量级的codeagent cli 的python 项目 ，并将开发支持tui 以方便使用的codeagent cli的项目

该项目的初始代码 是由我 进行编写，其来源于 claude code的项目（ E:\code\agentcli\claude-code 源码目录） 

基于此 准备使用python 语言打造一个属于我自己的codeagent cli 的项目， 为此我还额外下载了一个同样是python项目编写的codeagent开源项目（ 源码目录为 E:\code\agentcli\hermes-agent ） 这将作为你在本代码仓开发的核心参考

## 你的工作要求

主要参考 hermes-agent项目的代码 其中最最主要的是打通前端实现（ 因为我实在不懂前端代码 尤其是python代码）

你需要在现有的基础上参考hermes-agent的架构 和 claude code的架构，给我实现一个轻量化的 五脏俱全的（mcp, skills, 各种该有的tool ，前后端解耦 应该都是事件机制吧） ，

对于后端，其逻辑也将为按照hermes agent的逻辑 

你将不间断的执行，直到你可以实现出来这么一个类似hermess agent的轻量化的codeagent cli 或 tui的功能出来， 

**这是一个超长的任务 你必须保证你有记录中间的文档 总体规划 各个模块的设计 并允许你在后续实现阶段 修改前面的设计**
注意： 该过程中可能你会不断的check两边的代码， 分析设计实现，将你的中间产物，包括
- plan 任务实现规划 大纲
- design 各个模块的设计，与hermess或claude code的差异 等
- 等还有别的需要的文档
写在 doc目录下（ 这需要一起上传到github)

ai-work将作为一个你自由发挥的 你随意摆放任意文件 记忆的地方
